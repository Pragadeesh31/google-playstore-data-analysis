<h1>📊 Google Play Store Data Analysis</h1>

<p>
This project performs an in-depth <b>data analysis of applications</b> from the 
<b>Google Play Store dataset</b> to uncover insights about app categories, ratings, installs, and user feedback.
The objective of this project is to demonstrate <b>data cleaning, exploratory data analysis (EDA), and data visualization</b> using Python.
</p>

<hr>

<h2>📌 Project Overview</h2>
<p>
The dataset contains information about thousands of mobile applications including 
<b>category, rating, installs, price, and user reviews</b>. 
This project explores patterns and trends within the dataset to understand 
<b>app popularity, user engagement, and market distribution</b>.
</p>

<h2>🔍 Key Analysis Performed</h2>
<ul>
<li>Analysis of <b>top app categories</b> on the Play Store</li>
<li>Distribution of <b>Free vs Paid applications</b></li>
<li><b>Rating distribution</b> of apps</li>
<li><b>Sentiment analysis</b> of user reviews</li>
<li><b>App installs by category</b></li>

<h2>🎯 Key Insights</h2>
<ul>
<li>Most apps available on the Play Store are <b>Free applications</b>.</li>
<li>Categories such as <b>Family, Games, and Tools</b> dominate the platform.</li>
<li>The majority of apps have ratings between <b>4.0 and 4.5</b>.</li>
<li>Some categories contribute significantly to <b>higher install counts</b>.</li>
</ul>
